Exercise 1

Data

This exercise needs no data. You can flip a coin to create your own data set.

The Bayesian coin flip

Implement Bayesian updating for a coin flip experiment that tries to find the true probability \(\theta\) of heads.

  1. What is a suitable likelihood \(p(y\,|\,\theta)\)?
  2. What is the parameter \(\theta\) of that likelihood?
  3. How can you encode your prior belief about \(\theta\) in a distribution?
    • Possible values of \(\theta\)?
    • Areas of high mass?
    • Use R to sample and visualize some distributions.
  4. How does the result change if you increase the number of observations N?
  5. Optional:
    • Compute the Maximum Likelihood Estimate (MLE)
    • Compare the MLE with mode of the posterior distribution (aka. MAP)
    • When do MLE and MAP differ?
    • Can you make MLE and MAP equal?

Hints

Bayesian update from the example

N = 20
y = 14

theta = seq(0, 1, by=0.01)
prior = dbeta(theta, 3, 2)
likelihood = dbinom(y, N, theta)
joint = prior * likelihood

posterior = joint / sum(joint)

print(cbind(theta, prior, posterior)[1:10, ])

Visualizing prior and posterior

prior = prior / sum(prior)
plot(theta, posterior, type="l", col="orange", lwd=5, xlim=c(0,1), xlab="theta", ylab="", axes=FALSE)
lines(theta, prior, col="darkblue", lwd=5)
legend("topleft", legend=c("Prior", "Posterior"),
       col=c("darkblue", "orange"), lty=1, cex=1.7, lwd=10)
axis(1, pos=0)
